# Astrology Daily Engine

如何把"今天"读成真实的占星变量，让每一天的运势不一样。这些变量由运行时的确定性引擎计算（`src/lib/fortune/astro-day.ts`），并以"当日星象事实"注入生成提示。这里是给模型的**解读规则**：把确定性事实翻译成有画面感、非确定性的内容。

> 重要：这些变量是确定性输入，不是随机灵感。同一天、同一星座永远拿到同一组事实。你的任务是**解读**它们，而不是另编一套。

## 注入的当日变量（来自引擎）

- `日期 / 星期`
- `当日主行星(星期)`：星期对应的传统主宰行星与能量。
- `月相`：八相之一 + 它的行为含义。
- `太阳季节背景`：当前太阳所在星座（所有人共享的季节基调）。
- `目标星座` 与 `星座画像`：元素、模式、守护星、核心驱动、阴影。
- `今日侧重域`：事业 / 财运 / 感情 / 自我（由日期+星座确定性轮换，**决定今天主要谈哪个生活面**）。
- `情绪基调`：一句确定性的情绪底色。

## 月相 → 行为基调

| 月相 | 基调 | 适合 | 不适合 |
|---|---|---|---|
| 新月 | 起念 | 定一个小目标、开个头 | 急着要结果 |
| 蛾眉月 | 萌芽 | 给新计划耐心和保护 | 自我怀疑 |
| 上弦月 | 推进 | 顶着阻力做一个具体决定 | 回避冲突 |
| 盈凸月 | 打磨 | 调整细节、补强 | 推倒重来 |
| 满月 | 放大 | 看见、释放、庆祝 | 情绪上头下结论 |
| 亏凸月 | 消化 | 复盘、分享、感恩 | 又揽新摊子 |
| 下弦月 | 做减法 | 放下一个不再服务你的东西 | 硬撑 |
| 残月 | 收尾 | 休息、清空、留白 | 强行启动 |

## 星期主行星 → 能量

- 星期日·太阳：自我与休整。
- 星期一·月亮：情绪与节奏。
- 星期二·火星：行动与推进（别硬冲）。
- 星期三·水星：沟通与信息（把话说清楚）。
- 星期四·木星：扩展与机会（开放但有边界）。
- 星期五·金星：关系与价值（关照喜欢的人和事）。
- 星期六·土星：边界与收尾（把责任落地）。

## 今日侧重域 → 内容主轴

引擎会确定性地给出今天主谈哪个域。**必须**围绕这个域写，避免每天都写成同一个主题（这正是过去内容同质化的根源）。

- 事业：拖延的任务、模糊的边界、跨时区协作、自我证明的疲惫。
- 财运：账单、订阅、AA、汇率、跨境手续费、冲动消费、奖励自己。
- 感情：没说出口的话、暧昧的进退、孤独、报喜不报忧。
- 自我：精力管理、自我苛责、休息的负罪感、身份切换的累。

## 合成步骤

1. 读 `目标星座` 的核心驱动与阴影（见 `astrology-signs.md`）。
2. 读 `今日侧重域` → 确定主轴。
3. 用 `月相` 定"今天适合推进还是收口"。
4. 用 `星期主行星` 给一个能量色彩。
5. 用 `情绪基调` 校准语气。
6. 落成：一个反差 hook + 今日关键词 + 一个海外受众场景 + 一个非确定性的小动作。

### 示例（确定性事实 → 内容方向）

事实：目标星座=摩羯座；侧重域=自我；月相=下弦月；星期六·土星；情绪基调="疲惫但不想停"。

方向：摩羯座习惯用产出衡量自己（核心驱动+阴影）→ 自我域 + 下弦月做减法 + 土星收尾 → 「今天不是再多扛一点的日子，是允许自己放下一件事的日子。」关键词「松手」，动作"把一个你硬撑着的待办，今天正式标记为‘明天再说’"。

## 边界

- 永远非确定性表达（"今天更适合 / 你可能会注意到 / 今天的关键词是"）。
- 不因为是满月/下弦就制造吉凶恐惧。
- 月相、星期、星座只是**情绪与行为的隐喻**，不是命运开关。
