# X Thread Patterns

Use this for `outputType=thread` or `both`.

## Length

Final thread length: 5-8 items.

## Roles

Each item must have one role:

- hook
- emotional context
- concrete scene
- fortune interpretation
- practical action
- ritual
- CTA

## Suggested Progression

1. hook: one strong thesis.
2. emotional context: name the hidden feeling.
3. concrete scene: use specific overseas-life or work-life scenes.
4. fortune interpretation: decode the symbolic image.
5. practical action: one thing to do today.
6. ritual: tiny symbolic action.
7. CTA: natural comment prompt or series invitation.

## Thread Quality Bar

Every item should be independently readable. Avoid one item that only says "继续看下一条". Do not repeat the same sentence with minor changes.
