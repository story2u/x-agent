# Hook Patterns

Generate 5 hook options before final writing. The final hook should stop scrolling without promising outcomes.

## Contrarian Hook

Pattern: negate the generic fortune expectation, then replace it with a grounded insight.

Example patterns:

- "今天的财运，不一定是多进一笔钱，而是少漏一笔钱。"
- "今天不是等贵人的日子，是把账算清楚的日子。"
- "好运没有突然降临，它先藏在你愿意复核的那一笔里。"

## Scene Hook

Pattern: start inside a concrete scene.

Example patterns:

- "如果你这周已经第三次在咖啡、外卖和打车之间心软，今天先别骂自己。"
- "打开信用卡账单前那几秒，其实就是今天的财运入口。"

## Confession Hook

Pattern: create account personality by admitting a relatable tension.

Example patterns:

- "我越来越觉得，成年人的财运不是会不会发财，而是有没有力气面对细节。"
- "有些钱不是被花掉的，是被逃避掉的。"

## Mystical Image Hook

Pattern: use one symbolic image, then decode it into behavior.

Example patterns:

- "今天的画面像一个漏风的钱袋：不是要你抓更紧，而是先找到小洞。"
- "今天像雾散到一半，方向能看见了，但还不适合猛踩油门。"

## Practical Warning Hook

Pattern: warn gently, never threaten.

Example patterns:

- "今天先别在情绪高点转账、下单或答应请客。"
- "如果一个决定让你必须立刻付款，先给自己十分钟。"

## Avoid

- "今日运势关键词："
- "今天你的财运是："
- "今日财运不错"
- "一定发财"
- "马上转运"

## Weak Hook Rewrites

Weak:

```text
今日财运关键词：补漏。
```

Why weak:

- no conflict
- no audience scene
- no pause or tension
- sounds like a generic almanac line

Rewrite:

```text
今天的财运，不是让你暴富。

而是让你发现：
你不是赚得太少，
是有几个小洞，正在安静地漏钱。
```

Weak:

```text
今日事业运不错，适合努力工作。
```

Why weak:

- too generic
- no emotional truth
- gives no concrete action

Rewrite:

```text
今天的事业运，不是叫你再多扛一点。

而是提醒你：
有些任务不是你做不好，
是边界一开始就没被说清楚。
```

Weak:

```text
今天适合整理财务。
```

Why weak:

- practical but not scroll-stopping
- lacks symbolic image
- lacks reader recognition

Rewrite:

```text
打开信用卡账单前那几秒，
其实就是今天的财运入口。

你不需要责怪自己，
只需要把一个小洞先看清楚。
```
