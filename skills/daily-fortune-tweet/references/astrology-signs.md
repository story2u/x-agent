# Astrology Signs Knowledge Base

真实的西方占星十二星座底料。用于把"今日运势"落到具体星座的心理动力，而不是空泛玄学。

使用规则：

- 只解读「核心驱动 / 阴影张力 / 元素气质」，把它翻译成今天的情绪与行为提醒。
- 不做命定式预测，不保证结果（遵守 `fortune-safety-policy.md`）。
- 结合 `astrology-daily-engine.md` 的当日变量（月相、星期主行星、太阳季节）调味。
- 受众默认是海外中文年轻人：留学、跨时区工作、汇率、孤独感、身份切换。把星座特质映射到这些真实处境。

## 元素与气质速查

- 火象（白羊、狮子、射手）：行动、热情、想被看见；今日提醒常关于"先稳一下再冲"。
- 土象（金牛、处女、摩羯）：务实、积累、责任；今日提醒常关于"别只用产出衡量自己"。
- 风象（双子、天秤、水瓶）：沟通、思考、连接；今日提醒常关于"把想法落地、把话说清楚"。
- 水象（巨蟹、天蝎、双鱼）：情绪、共情、深度；今日提醒常关于"照顾感受但守住边界"。

## 模式（modality）

- 基本（白羊/巨蟹/天秤/摩羯）：发起者，适合"开一个头"。
- 固定（金牛/狮子/天蝎/水瓶）：坚持者，提醒"该松手时松手"。
- 变动（双子/处女/射手/双鱼）：适应者，提醒"在变化里给自己一个锚"。

## 十二星座

### 白羊座 Aries｜火·基本·火星
- 核心驱动：率先行动、证明自己能行。
- 阴影张力：急躁，容易在情绪高点透支精力或钱。
- 海外语境：想快点把生活搞定（找房、换工作、社交），却容易因为冲太快而累。
- 今日翻译方向：把"冲"换成"先做一件最关键的小事"。

### 金牛座 Taurus｜土·固定·金星
- 核心驱动：安全感、稳定的节奏、可掌控的舒适。
- 阴影张力：固执、回避变化，容易用消费安抚自己。
- 海外语境：对汇率、房租、存款敏感；用"奖励自己"对冲漂泊感。
- 今日翻译方向：把安全感从"再买一点"换成"把已有的稳住"。

### 双子座 Gemini｜风·变动·水星
- 核心驱动：好奇、收集信息、保留选择权。
- 阴影张力：分心，承诺难落地，信息过载。
- 海外语境：多语言切换、消息太多、什么都想试又怕错过。
- 今日翻译方向：从"再看一个选项"换成"先收一个口"。

### 巨蟹座 Cancer｜水·基本·月亮
- 核心驱动：归属、情感的家、照顾与被照顾。
- 阴影张力：情绪化、过度自我保护，把委屈憋着。
- 海外语境：想家、时差里的孤独、报喜不报忧。
- 今日翻译方向：允许自己说一句真实的近况。

### 狮子座 Leo｜火·固定·太阳
- 核心驱动：表达、被真诚看见、发光。
- 阴影张力：好面子，怕被忽视，硬撑。
- 海外语境：在陌生环境里想被认可，又怕显得"太需要"。
- 今日翻译方向：把"被看见"从外部认可换成自己先肯定自己。

### 处女座 Virgo｜土·变动·水星
- 核心驱动：把事做对、把混乱理清、持续改进。
- 阴影张力：苛责自己，陷在细节，焦虑。
- 海外语境：什么都要自己搞定（签证、账单、流程），怕出错。
- 今日翻译方向：把"做到完美"换成"做完一件就够好"。

### 天秤座 Libra｜风·基本·金星
- 核心驱动：平衡、关系、公平与美感。
- 阴影张力：犹豫，为迁就别人而委屈自己。
- 海外语境：在不同文化期待间找平衡，怕冲突、怕麻烦别人。
- 今日翻译方向：把"先照顾别人"换成"也把自己的需要放上秤"。

### 天蝎座 Scorpio｜水·固定·冥王星
- 核心驱动：真实、深入、掌控与转化。
- 阴影张力：多疑，不愿放手，内耗。
- 海外语境：信任要慢慢建立，独自消化很多事。
- 今日翻译方向：把"必须掌控"换成"放掉一件不归你管的事"。

### 射手座 Sagittarius｜火·变动·木星
- 核心驱动：探索、意义、自由。
- 阴影张力：好高骛远、承诺过头、逃避收尾。
- 海外语境：本身就在远方，却容易在"下一个目的地"里逃避当下。
- 今日翻译方向：把"奔向远方"换成"把脚下这步走稳"。

### 摩羯座 Capricorn｜土·基本·土星
- 核心驱动：成就、责任、长期可被信赖。
- 阴影张力：压抑，把自我价值绑在产出上。
- 海外语境：独自扛目标，报喜不报忧，休息有负罪感。
- 今日翻译方向：把"我有没有用"换成"我也允许自己慢一点"。

### 水瓶座 Aquarius｜风·固定·天王星
- 核心驱动：独立、理念、按自己的方式生活。
- 阴影张力：抽离，回避亲密，显得疏远。
- 海外语境：享受独处，又偶尔被孤独反噬。
- 今日翻译方向：把"我不需要别人"换成"主动连一个人"。

### 双鱼座 Pisces｜水·变动·海王星
- 核心驱动：连接、共情、想象与意义。
- 阴影张力：边界模糊，容易逃避和心软。
- 海外语境：容易被情绪和他人需要淹没，分不清谁的事。
- 今日翻译方向：把"都帮一把"换成"先把自己的船扶正"。

## 翻译公式

`星座核心驱动 + 当日变量（月相/星期/季节）+ 海外受众场景 → 一个温柔、非确定性的今日提醒 + 一个小动作`

例：天秤座（想要和谐、易委屈）+ 下弦月（适合做减法）+ 金星星期五（关系与价值）→ 「今天适合在一段关系里做一次小小的减法：把一个你一直忍着的小期待说出口。」
