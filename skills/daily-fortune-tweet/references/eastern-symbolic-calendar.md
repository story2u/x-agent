# Eastern Symbolic Calendar (MVP)

## Role

为今日运势内容提供东方象征语言。**这不是确定性预测系统，也不是传统黄历。** 所有传统符号都必须翻译成现代、非确定性的行动建议，配合 Seth 意识框架把主动权还给读者。

运行时由 `src/lib/fortune/eastern-day.ts` 计算并注入这些变量（均带来源标注）：生肖年、节气、五行当令、季节行动。它们是**近似/象征**的，不要当成精确命理推算。

## Five Elements → Modern Actions

把五行翻译成现代行动，不要讲玄学术语堆砌。

- 木：生发、计划、启动、学习、新芽
  - 现代行动：开个小头、把一个想法写下来、约一次沟通、报名一件想学的事
- 火：表达、曝光、情绪、热度、冲动
  - 现代行动：把话说清楚、把作品发出去；但别在情绪高点下决定或消费
- 土：稳定、边界、整理、承载（过渡元素）
  - 现代行动：核对账单、清理桌面、稳定日程、把承诺写下来
- 金：收口、规则、判断、取舍
  - 现代行动：取消、编辑、删减、设边界、复核细节、把一件事正式结束
- 水：流动、信息、观察、直觉
  - 现代行动：先听、再等一等、收集信息、复盘消息再回应

## Solar Terms（节气）作季节氛围

节气只作**季节基调与隐喻**，不作吉凶判断。示例：

- 惊蛰 → 蛰伏的事开始动，适合把想法说出口
- 清明 → 清理与记得，适合整理与告别
- 立夏 → 能量上升，但节奏比冲劲重要
- 芒种 → 该收的先收，别贪多
- 白露 → 转凉，照看边界与身体
- 霜降 → 收敛，放下一个不再服务你的东西
- 冬至 → 转折点，安静重启

## Zodiac Year（生肖年）

生肖年用作**长期氛围与自我叙事**的轻背景，不要写成「你今年一定如何」。它按立春近似计算，可能与某些春节流派差一年——所以只作风格点缀，不作断言。

## Writing Rules

- 把传统符号翻译成现代行动，**不要照搬**「宜嫁娶、忌出行、诸事不宜」这类黄历格式。
- 不写「冲煞」「凶日」「破财日」这类制造恐惧的表达。
- 东方象征与西方星象、Seth 框架并列时，挑 1-2 个最相关的用，别全部堆上。
- 节气/五行是隐喻，最终要落到一个具体、温和、非确定性的小行动。

示例转译：

- 传统「金日宜收」→ 现代「今天适合做一次收口：取消一个不需要的订阅，或把一段拖着的对话结束掉」。
- 传统「水日宜静」→ 现代「今天信息有点多，先听、先看，重要的回复可以慢半拍」。
