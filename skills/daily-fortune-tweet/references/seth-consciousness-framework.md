# Seth Consciousness Framework for Daily Fortune

## Role

This reference is the **meaning layer**, not a prediction engine. It is an internal
writing lens inspired by Seth-style consciousness ideas — it is NOT a quotation of
any copyrighted text, and content must never claim to authoritatively channel or
cite "赛斯". Do not write "赛斯说今天会…".

象征系统（星座、月相、生肖、五行、节气、塔罗）是镜子。本层把象征翻译成：

- 注意力（放在哪里）
- 信念（哪个正在影响体验）
- 概率线 / 选择点（现在可以拨向哪边）
- 情绪信号（情绪在提示什么，不是命令）
- 当下力量点（改变发生在现在）
- 一个小行动（给下一条分支留一个更稳的入口）

目标：减少恐惧，增加能动性，把主动权还给读者。

## Core Principles

1. 未来是概率性的，不是固定的。
2. 当下是力量点 —— 改变从现在的注意力和选择开始。
3. 注意力组织经验：你关注什么，就更容易经验到什么。
4. 信念塑造一个人会注意到、会选择的概率。
5. 情绪是信息，不是命令，也不是审判。
6. 外部象征是内在组织状态的镜子，不是命运开关。
7. 一个有意识的小行动，可以轻轻拨动下一条概率线。
8. 一条好的运势，应该让读者更有能动性，而不是更害怕。

## Writing Translation（必须这样改写）

不要写：今天一定会破财。
要写：今天更适合看见一个正在悄悄消耗你的模式。你不需要害怕它，只需要把注意力先收回来。

不要写：好运会降临。
要写：好运更像是你把注意力从焦虑里拿回来之后，终于看见了一个原本被忽略的机会。

不要写：这是命中注定。
要写：这更像是一个选择点。你可以从这里，把概率线轻轻拨向更稳的方向。

不要写：今天必有贵人。
要写：今天适合把需要帮忙的事说得更清楚一点，你会更容易接住向你靠近的人。

## Fortune Content Usage

生成今日运势时：

- 用这套意识框架解释「为什么这个象征值得注意」，而不是预测结果。
- 把象征（月相 / 星座 / 五行 / 节气）落到注意力、信念、选择点、小行动。
- 涉及财运：强调行动提醒 / 注意力回收 / 概率选择，而不是发财承诺。
- 最终至少体现「注意力 / 信念 / 概率线 / 当下力量点 / 选择点 / 小行动」中的一项。
- 给读者一个今天就能做的小动作。
- 语气温柔但清醒，不说教、不灌鸡汤。

## Preferred Phrases

- "这不是预言，更像是一个提醒。"
- "今天的重点不是控制未来，而是把注意力收回来。"
- "你正站在一个选择点上。"
- "情绪不是命令，它只是信号。"
- "概率线不是固定的，它会被你的注意力和选择影响。"
- "当下，就是你能用力的那个点。"
- "今天的小动作，是给未来的自己留一个更稳的入口。"

## Avoid（宿命化表达，禁止）

- "命中注定"
- "无法改变"
- "一定会发生"
- "必然失去"
- "不照做就会倒霉"
- "劫数 / 在劫难逃"
