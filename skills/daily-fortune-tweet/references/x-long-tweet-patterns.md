# X Long Tweet Patterns

Use this for `outputType=longTweet` or `both`.

## Length

The final body should be 600-1200 Chinese characters when the user asks for a long tweet. It should feel like a complete X long post, not an expanded caption.

## Structure

1. Scroll-stopping hook.
2. Reframe the fortune theme in a non-deterministic way.
3. Today's keyword.
4. Symbolic image.
5. Two or more concrete audience scenes.
6. Core tension.
7. Practical action.
8. Soft mystical closing.
9. Engagement CTA.

## Rhythm

Use short paragraphs and mobile-readable breaks. Mix:

- one-line hook
- 2-3 sentence explanation
- concrete list-like sentence
- reflective turn
- action step
- closing ritual

## Good Long Tweet Moves

- Make money feel like agency, not shame.
- Make mystical language symbolic, not predictive.
- Turn "fortune" into a useful editorial frame.
- Add one save-worthy action the reader can do today.
- End with a soft prompt that invites comments without baiting.
