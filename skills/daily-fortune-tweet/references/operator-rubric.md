# Operator Rubric

Score draftV1 before final output. Every score is 1-5.

## Scoring Criteria

- hookStrength: Does the first line create tension or curiosity?
- specificity: Are there concrete scenes, not generic adjectives?
- audienceFit: Does it sound written for the named audience?
- emotionalResonance: Does it name a real feeling without melodrama?
- shareability: Would someone send this to a friend because it feels true?
- saveWorthiness: Is there a useful checklist, action, or reminder?
- safety: Does it avoid deterministic prediction and professional advice?

## Rewrite Rule

If any score is below 4, rewrite once before producing `final`.

Common rewrite directions:

- weak hook: replace generic opening with contrast or scene.
- low specificity: add two concrete scenes.
- low audience fit: use the user's location/life context.
- low resonance: name the hidden tension more directly.
- low save-worthiness: add one concrete action.
- low safety: remove guaranteed outcomes and add entertainment/reflection framing.
