# 开发计划

## 当前 MVP 状态

- 主操作入口已切换为本地 CLI/TUI：`npm run tui`。
- TUI 是单输入框 client：普通输入生成文本，其他操作通过 `/` slash commands。
- TUI 已支持 `/skills`、`/skill`、`/tone`、`/output`、`/audience`、`/goal`、`/constraints`、`/model`、`/last`、`/history`。
- X/Twitter 文本生成 agent 已接入 pi agent loop。
- 模型调用默认使用 `openai-codex`，可通过本地 ChatGPT Plus/Pro OAuth credentials 连接；也支持 `PI_PROVIDER=deepseek` 通过 DeepSeek API 调用。
- Skills 已切换为本地 `skills/*/SKILL.md`，不再以 D1 为 source of truth。
- 已内置 `twitter-launch-creative` 和运营级 `daily-fortune-tweet`。
- Web 登录、Web Review Gate、后台审批流已退出当前 MVP 主路径。
- MVP 不提供图片生成、图片方向、alt text 或 media upload。

## 已完成

- 移除全部 Next.js API、D1、Durable Object、Cloudflare 部署和 Web 登录 / RBAC 代码。
- 凭据改为 `.env` / 环境变量；TUI 启动时加载 `.env`。
- 模型错误（`stopReason: error`）直接暴露真实原因，不再被保底模板掩盖。
- `chatgpt:oauth` 输出单行凭据；`maxTokens` 提到 8192 并支持 `PI_MAX_TOKENS`。
- `daily-fortune-tweet` 升级为 audience insight、angle mining、hook lab、operator critique、rewrite loop、engagement plan 的运营流水线。
- 本地 skill 支持 `references/*.md` 加载到 runtime prompt。
- 新增 Daily Fortune eval-style tests。
