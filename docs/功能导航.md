# 功能导航

按功能域导航到代码、数据和文档。

## CLI / TUI Client

- 文档：`features/cli-tui.md`
- TUI：`scripts/x-agent-tui.ts`
- Scripts：`package.json` 的 `tui`、`cli`
- Local skills：`skills/*/SKILL.md`
- Skill loader：`src/lib/skills/local-skills.ts`
- Agent：`src/lib/pi-agent.ts`

## Text Agent

- 文档：`features/text-agent.md`
- Agent：`src/lib/pi-agent.ts`
- 类型：`src/lib/types.ts`
- 测试：`src/lib/__tests__/pi-agent.test.ts`、`src/lib/__tests__/validation.test.ts`

## Local Skills / Workspace Harness

- 文档：`features/workspace-harness.md`
- Skill files：`skills/twitter-launch-creative/SKILL.md`、`skills/daily-fortune-tweet/SKILL.md`
- Loader：`src/lib/skills/local-skills.ts`
- Parser：`src/lib/skills/parse-skill.ts`
- Validator：`src/lib/skills/validate-skill.ts`
- 测试：`src/lib/__tests__/skills.test.ts`

## Daily Fortune Skill

- 文档：`features/workspace-harness.md`、`features/text-agent.md`
- Skill file：`skills/daily-fortune-tweet/SKILL.md`
- Runtime selection：`src/lib/skills/local-skills.ts`
- Agent recovery/rendering：`src/lib/pi-agent.ts`、`scripts/x-agent-tui.ts`
- 测试：`src/lib/__tests__/skills.test.ts`

## ChatGPT Plus/Pro Model OAuth

- 文档：`features/model-oauth.md`
- TUI：`/model` in `scripts/x-agent-tui.ts`
- Runtime credentials：`src/lib/pi-credentials.ts`
- OAuth script：`scripts/chatgpt-oauth-login.mjs`
- 凭据来源：`.env` 的 `OPENAI_CODEX_OAUTH_CREDENTIALS` / `OPENAI_CODEX_ACCESS_TOKEN` / `OPENAI_API_KEY`
