## Daily Fortune Pipeline（星座）

- 文档：`features/text-agent.md`、`docs/数据流程.md`
- 星象引擎（确定性）：`src/lib/fortune/astro-day.ts`；provenance 类型：`src/lib/fortune/types.ts`
- 5 段推理 pipeline：`src/lib/fortune/pipeline.ts`
- FortuneContext（western/eastern/seth/creative 四层）：`src/lib/fortune/context.ts`；东方时间层：`src/lib/fortune/eastern-day.ts`；trace 类型在 `src/lib/fortune/types.ts`
- 正文边界 guard：`src/lib/fortune/public-surface.ts`（泄漏短语 + 技术黑话）、`src/lib/fortune/field-present.ts`（requiredFields 解析）
- 开头 guard：`src/lib/fortune/pipeline.ts` 的 `normalizeLongTweetOpening`
- 路由入口：`src/lib/pi-agent.ts` 的 `generateTwitterCreative`（`skillSlug === "daily-fortune-tweet"` 分支）
- 命理底料 references：`astrology-signs.md`、`astrology-daily-engine.md`、`eastern-symbolic-calendar.md`、`seth-consciousness-framework.md`、`public-post-boundary.md`、`playful-fortune-voice.md`（均在 `skills/daily-fortune-tweet/references/`）
- 真 eval：`scripts/eval-fortune-run.ts`（`npm run eval:fortune`）
- 测试：`src/lib/fortune/__tests__/astro-day.test.ts`、`src/lib/fortune/__tests__/pipeline.test.ts`
